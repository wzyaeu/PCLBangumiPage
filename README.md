# PCL_Bangumi_Page

一个使用[**Bangumi API**](https://bangumi.github.io/api/)的[**PCL**](https://github.com/Meloong-Git/PCL/)主页，展示Bangumi上的信息。

## 特点

- 每日放送
    展示Bangumi上每周的**每日放送**

    ![](https://raw.githubusercontent.com/wzyaeu/PCLBangumiPage/refs/heads/main/image/rm1.png)

    ![](https://raw.githubusercontent.com/wzyaeu/PCLBangumiPage/refs/heads/main/image/rm2.png)
    
- 动漫排行榜
    按照Rank排行的**动漫排行榜**

    ![](https://raw.githubusercontent.com/wzyaeu/PCLBangumiPage/refs/heads/main/image/rm3.png)
